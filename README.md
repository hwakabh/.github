# .github
GitHub Repository Scaffolder

## Features
- Git metafiles
  - gitignore (generic)
  - gitattributes
- GitHub builtins
  - README: this file
  - CODEOWNERS
  - SECURITY
  - CODE_OF_CONDUCT
  - LICENSE
- GitHub Issue/PR templates
- labeler (issues and PRs)
- Semantic PR
- Stale Issue
- (optional) default labels

- release-please
- renovate
